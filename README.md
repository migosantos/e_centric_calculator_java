# E-centric Calculator (Java)
Calculator (JAVA based)

## Building and Running JUnit Tests:

1. Open cmd, 
2. Go to project ROOT LOCATION
3. Execute the following maven command:

```sh
$ mvn clean install
```

## To run the actual application:

1. Open cmd, 
2. Go to project ROOT LOCATION
3. Execute the following maven command:

```sh
$ mvn exec:java
```
